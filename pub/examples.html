Freebody.js Examples

Freebody.js currently supports 2 types of diagrams:


Inclined Planes

Flat Planes

Both diagrams share much of the same functionality excluding those specific to the inclined plnae (Slope and plane highlighting). Obviously, there is more functionality than just displaying, so lets dive into some examples.


Each diagram comes with built-in input generators for key styles. In the inclined plane these include the ability to set the slope and the height. In the flat plane, the only built-in input is for width of the diagram. Below is an example for the inclined plane.


Feel free to mess around with the values. As a warning, extreme slopes and heights may make the model look odd!

There is also the ability to add custom height and width labels. These fall into 2 categories:
  • Number labels: If the height is set using a number label, the width will be automatically calculated. You are also able to add units which will globally apply to the width too
  • Text labels: If the height/width are set using a non-number, you can set custom values like "6x". If only the height is set, the width will be set using the formula width = height / tan(angle)
