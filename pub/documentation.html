Getting started:

To initialize a new diagram, we must run the following commands first:

//Make a new inclined plane with name "p1"
const FBD = new inclinedPlane("p1")
FBD.generate()

//Make a new flat plane with name "p2"
const FBD2 = new plane("p2")
FBD2.generate()

This generates the diagram with no force object on it, meaning we just have a plane/inclined plane with the default values for slope, height, and width.


To add force objects, use the following command:

FBD.addForceObject("box")
FBD.generateBox()


Functions for both diagram types

generate()

Params:
  • None

The generate function renders the diagram according to the current set parameters.

Note: if you have generated a box object on your model, use generateBox() instead!


generateBox()

Params:
  • None

The generateBox function renders the diagram according to the current set parameters.

Note: if you do not want a force object, do not use generateBox!


addForceObject(Object)

Params:
  • Object: String

Adds details to allow for generation. Object = "box" will add a box to the diagram.

Note: The only implemented objects are "box" and "square" which are both the same!


setLeftOffset(offset)

Params:
  • offset: Number

Moves the diagram offset pixels to the right of the canvas.

Note: This also changes the size of the canvas.


setUpOffset(offset)

Params:
  • offset: Number

Moves the diagram offset pixels towards the bottom of the canvas.

Note: This also changes the size of the canvas.


setMass(mass, unit)

Params:
  • mass: Any
  • unit: String (Optional)

Adds/updates the mass attribute of the force object. If mass and unit are set, the popup will display the mass as: "mass unit" (i.e. 45 kg). If unit is not set, the mass will be set to just the mass value, this is useful for setting the mass equal to a variable

Note: This will only be displayed in the popup


setForces(Forces)

Params:
  • Forces: Object

Sets any of the preset forces [Normal, Gravity, Right, Left, Down] to the assigned value. The passed in object must be in the form:
{"Normal": "Value", "Down": "Value", ...}
You are allowed to set any forces you choose among the preset forces offered

Note: This will only be displayed in the popup


allowPopup()

Params:
  • None

This allows the popup to be created when clicking on the plane (only for inclined plane) and the force object


disablePopup()

Params:
  • None

This stops the popup from being created when clicking on the plane (only for inclined plane) and the force object


Functions only in inclinedPlane


setHeightPx(height)

Params:
  • height: Number

Sets the pixel height of the plane

Note: This MUST be greater than 0, and should be greater than 50


setHeight(height, unit)

Params:
  • Height: Any
  • unit: String

sets the height label to the value given
